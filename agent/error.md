# Handle each error

## Could not load credentials from any providers

This error occurs when the frontend agent will be try to upload files to the backend local server, To fix this error, you need to set up the CC admin please follow below steps:

1. Go to the CC admin panel.
2. Click on the `Setting` tab.
3. Click on the `Files` tab.
4. Click on the `show_config` button(on the right of toolbar).
5. Change the `Provider` to `local`.
